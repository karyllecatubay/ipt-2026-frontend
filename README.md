# Final Project: Full-Stack Authentication System Deployment

A comprehensive Angular 21 boilerplate featuring authentication, user management, and role-based access control.

## Live Deployment
- **Frontend:** https://ipt-2026-frontendd.onrender.com
- **Backend:** https://ipt-2026-backend-ujcl.onrender.com
- **Swagger Docs:** https://ipt-2026-backend-ujcl.onrender.com/api-docs

## Features
- Email Sign Up and Verification
- JWT Authentication with Refresh Tokens
- Role Based Authorization (Admin & User)
- Forgot Password and Reset Password
- Admin Panel for managing accounts

## Getting Started

1. Install dependencies:
```bash
   npm install
```

2. Run development server:
```bash
   npm start
```

3. Build for production:
```bash
   npm run build
```

## Tech Stack
- Angular 21
- Node.js + TypeScript
- MySQL (Aiven)
- Bootstrap 5
- JWT Authentication
- Resend (Email Service)
- Deployed on Render
